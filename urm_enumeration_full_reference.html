Таблица 4 — нумерация всех команд МНР (обозначение Li)

Команды идут блоками по 4: Z(k), S(k), Tk, Jk — для k = 1, 2, 3, ... Где Tk = T(n,m) — k-я строка Таблицы 2; Jk = J(n,m,r) — k-я строка Таблицы 3.

Таблица 1 — нумерация строк из ℤ≥0, обозначение Li

Строки упорядочены: сначала по сумме компонент (весу), внутри одного веса — лексикографически. Все компоненты ≥ 1. Это и есть нумерация программ: строка (i1,...,id) задаёт программу Pr, и её команды — Li1, ..., Lid.

Таблица 2 — строки длины 2, обозначение mi (для команды T)

T(n,m) = Ti, где i — номер строки (n,m) в этой таблице.

Таблица 3 — строки длины 3, обозначение qi (для команды J)

J(n,m,r) = Ji, где i — номер строки (n,m,r) в этой таблице.

Справка: команды МНР

Z(n) — обнулить регистр Rn: Rn := 0
S(n) — увеличить регистр Rn на 1: Rn := Rn + 1
T(m,n) — скопировать Rm в Rn: Rn := Rm
J(m,n,q) — условный переход: если Rm = Rn, перейти к команде Lq; иначе к следующей. Если q > длины программы — остановиться.
Обозначения результата:
P(a1,a2,...) ↓ — программа P останавливается при входе (a1,a2,...), результат = значение R1
P(a1,a2,...) ↑ — программа не останавливается
fP(k)(a1,...,ak) — результат выполнения P на k аргументах
